📘 Blog Application (Django)

A full-featured blog web application built with Django that provides secure authentication, user interaction features, and a modern UI experience.

🚀 Features

👤 User Authentication

User registration and login
OTP-based email verification
Forgot password and reset password system
Secure session management

🧑 User Profile

View and update profile information
Change username
Update email address (with verification)
Change password securely

📝 Blog Articles

Create, read, update, and delete articles
Modern and responsive UI layout
Organized article feed

💬 Comments System

Users can comment on articles
One user can comment on multiple articles
Edit and delete own comments only

⭐ Save & Watch Later

Save articles for later reading
Add/remove articles from Watch Later list
Easy access to saved content

📩 Email Features

OTP verification emails
Password reset emails
Email update confirmation

🛠️ Tech Stack

Backend: Django (Python)
Frontend: HTML, CSS, Tailwind
Database: PostgreSQL 
Authentication: Django auth system + custom OTP flow
